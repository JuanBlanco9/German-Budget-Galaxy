  git clone https://github.com/JuanBlanco9/Budget-Galaxy.git germany-ngo-map
  cd germany-ngo-map

  npm install   # installs xlsx, @anthropic-ai/sdk
  # Python is only needed for a few legacy parsers (scripts/build_uk_trees_all.py etc)

  export ANTHROPIC_API_KEY="<set via operator's password manager>"

  password manager OR regenerate them:

    - Companies House: developer.company-information.service.gov.uk
      (manage-applications → create REST API key)
  The Companies House key was rotated in April 2026 — any previously
  documented value is no longer valid, regenerate before use.

Step 4 — If you need the raw spend CSVs (for re-processing or adding new councils):
  scp -i ~/.ssh/id_agro_intel -r \

9. CRITICAL RULES (carried forward from prior sessions)
==================================================

1. NEVER use XLSX sheet_to_csv — breaks on amounts with commas like "1,234.56".
   Always use sheet_to_json + manual CSV writing with proper quoting.
   (Fix was needed multiple times in this session.)

2. NEVER modify tree JSON files on disk to restructure them. The Multiverse view
   is extremely fragile to structural changes. Restructure in-memory in the frontend
   (see _loadUSCombinedTree, _loadUKCombinedTree, _loadFRCombinedTree).

3. NEVER fabricate supplier amounts or invent data to make ratios match. If a fix
   leaves a discrepancy, document it as a known limitation. The user explicitly
   values honest disclosure over polished-but-synthetic numbers.

4. METADATA vs TREE CHILDREN: When two datasets don't reconcile (e.g. Spend >£500
   vs MHCLG net current expenditure), attach as metadata on the parent node, not as
   tree children. Never force scaling to make totals match.

5. Tree integrity: after any injection, verify with a walker:
   let d=0; function w(n){if(n.children){
     const s=n.children.reduce((a,c)=>a+(c.value||0),0);
     if(Math.abs(s-n.value)>0.5)d++;n.children.forEach(w);}}
   Inject script outputs "Tree integrity: ✓ unchanged" when no new drifts.
   The 25 pre-existing drifts at OSCAR L5 department level are expected.

6. Deploy only after validating JS syntax. Always keep a backup of working frontend
   in data/backups/index.html.working.YYYY-MM-DD_HHMM.

7. Council name matching in inject_council_spend_metadata.js is loose — it uses
   startsWith + word-boundary matching. This is why "Kent CC" supplier data also
   attaches to "Kent Police" and "Kent Combined Fire and Rescue Authority" nodes.
   This is intentional — they're real entities in the tree.
